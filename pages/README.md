## About This Hub

The `pages/` directory contains all centralized, non-paper-specific documentation for the [docxology/docxology](https://github.com/docxology/docxology) repository. Each page includes:

- **YAML frontmatter** for SEO and static site generation
- **Navigation header** for seamless cross-page traversal
- **Standardized footer** with return-to-home link

### Related Directories

- **[papers/](../papers/)** — 108 per-paper folders with README, AGENTS, and SKILL files
- **[code/](../code/)** — Source code, orchestrators, and test suites
- **[docs/](../docs/)** — Repository-wide operational documentation
- **[works/](../works/)** — Generated HTML landing pages for every bibliography row
- **[reports/](../reports/)** — Source snapshots, reconciliation, external-link triage, live verification, accessibility, browser smoke, asset-size, and visual QA artifacts
- **[data/](../data/)** — Structured exports for works, software, people, organizations, claims, reconciliation, enrichment, and generated-artifact metadata
- **[art/](../art/)** — Visual art gallery assets
