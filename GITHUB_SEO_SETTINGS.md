## 4. Enable GitHub Pages

> Go to [github.com/docxology/docxology/settings/pages](https://github.com/docxology/docxology/settings/pages)

- **Source**: Deploy from a branch
- **Branch**: `main`
- **Folder**: `/ (root)`
- Click **Save**

The `index.html` in this repo will be served at `https://docxology.github.io/docxology/`
