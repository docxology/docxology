## Abstract

A technical and legal reference to California's public-records ecosystem, anchored by the California Public Records Act (CPRA), recodified by Assembly Bill 473 effective January 2023 to Government Code §§ 7920.000 et seq., and extended in practice through state and local open-data portals, request platforms, API frameworks (CKAN, Socrata, ArcGIS), metadata schemas, intergovernmental data-sharing surfaces, and exemption rules.

## Keywords

California Public Records Act · open data · CKAN · Socrata · ArcGIS · cognitive security · civic technology
