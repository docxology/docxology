*A Computational Meta-Analysis of the Active Inference Literature (2000–2026)*

## Abstract

> This paper introduces a Literature Review Architecture (LRA) for systematic, scalable analysis of the Active Inference literature. The architecture integrates three innovations: (1) a scalable assertion extraction pipeline that identifies and structures empirical and theoretical claims from full-text papers; (2) a nanopublication framework that encodes extracted assertions as machine-readable, citable knowledge atoms; and (3) a citation-weighted hypothesis scoring system that ranks hypotheses by their evidential support across the citation network. Applied to the Active Inference literature spanning 2000–2026, the LRA provides a quantitative landscape of the field's theoretical commitments, empirical grounding, and open questions. The accompanying open-source repository provides all code, data, and reproducible analysis pipelines.

## Key Contributions

- A Literature Review Architecture (LRA) combining assertion extraction, nanopublications, and citation-weighted hypothesis scoring into a unified, reproducible pipeline.
- Scalable assertion extraction from Active Inference full-text papers with structured claim representation.
- Nanopublication encoding of extracted assertions for machine-readable, semantically rich knowledge representation.
- Citation-weighted hypothesis scoring enabling evidence-based ranking of theoretical claims across the 2000–2026 Active Inference literature.

## Methods

- Full-text retrieval and preprocessing of Active Inference literature (2000–2026) via open-access repositories.
- Assertion extraction using NLP pipelines targeting hypothesis-bearing sentences; structured encoding as nanopublications.
- Citation network analysis for weighting hypothesis support; scoring metrics for ranking claims by cumulative evidential mass.
- Reproducible pipeline available at https://github.com/ActiveInferenceInstitute/act_inf_metaanalysis
